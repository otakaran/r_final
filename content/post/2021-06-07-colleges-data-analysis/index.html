Introduction

The purpose of this analysis is to understand employment trends by major.

The data-set selected is available online https://www.kaggle.com/tunguz/college-majors?select=all-ages.csv. We will be looking into the all-ages.csv file which shows employment information for all college graduates, regardless of age from the US census. This data was used in a FiveThirtyEight article.

dataset <- read.csv(here::here("data", "all-ages.csv"))

The dataset was imported locally and is some summary information to get acquainted with the data. The first column contains the exact name of the major, the second column contains 16 major “categories” which will be used thought this analysis. The categories are:

Basic Data Understanding

Let’s begin the basic analysis with some plots.

dataset %>%
  ggplot(aes(Employed, Major_category)) + geom_boxplot()

I took one look at this graph and immidiatly realize how useless it was. Because the number of people surveyed in each major was not constant, the results from the plot were meaningless. We will add another column which will have the percentage employed and then plot that.

dataset <- dataset %>%
  mutate(pct_employed = Employed / Total * 100)

dataset %>%
  ggplot(aes(pct_employed, Major_category)) + geom_boxplot()

Now this plot tells us much more. Immediately I notice the wide spread, and significantly skewed low employment in the education major category.

Next I want to look at a few majors individually and see how the employment distributes.

dataset %>%
  ggplot(aes(pct_employed)) + geom_histogram()
